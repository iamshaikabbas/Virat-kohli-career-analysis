Interactive Power BI dashboard analyzing Virat Kohli's international cricket career across Tests, ODIs, and T20Is with performance trends, opponent analysis, venue insights, and batting statistics.
# 🏏 Virat Kohli Career Statistics Analysis | Power BI Dashboard

## Overview

This Power BI project provides a comprehensive analysis of Virat Kohli's international cricket career across all three formats — Test, ODI, and T20I.

The dashboard explores batting performance, consistency, scoring patterns, strike rates, opposition-wise records, venue analysis, and career progression over time. By combining historical innings-level data with interactive visualizations, the project offers deep insights into one of the greatest modern-day cricketers.

---

## Project Objective

The objective of this project is to analyze Virat Kohli's international batting career and answer key performance-related questions such as:

- How has Virat Kohli performed across different formats?
- Against which teams has he scored the most runs?
- How has his performance evolved over time?
- What are his scoring patterns and consistency levels?
- Which venues and opponents bring out his best performances?
- How do strike rates and batting averages vary across formats?

---

## Dataset Information

The project uses innings-level batting data from Virat Kohli's international career across:

### ODI Cricket
- 262 ODI innings

### T20 International Cricket
- 114 T20I innings

### Test Cricket
- 180 Test innings

Each dataset contains:

- Runs Scored
- Balls Faced
- Minutes Batted
- Fours
- Sixes
- Strike Rate
- Batting Position
- Dismissal Type
- Opposition Team
- Match Venue
- Match Date
